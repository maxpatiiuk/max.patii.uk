# @maxpatiiuk/typescript-config

Shared TypeScript configurations for the `max.patii.uk` monorepo.

## Overview

This package provides base `tsconfig` files for different environments:

- **Base**: Strict mode, ESNext module, Bundler resolution.
- **DOM**: Adds DOM libs and experimental decorators for Lit.
- **Node**: Optimized for NodeNext module and resolution for tools like
  `static-site-forge`.
